Μέσα στον φάκελο source code υπάρχει το αρχείο run.bat που αποτελεί το εκτελέσιμο. Επίσης στον φάκελο sample υπάρχει ενα προτυπο αρχείο δεδομένων που μπορεί να χρησιμοποιήσει ο χρήστης για αρχικό dataset. 

Για την σωστή λειτουργία απαιτείται jdk-1.6.

Το project υλοποιήθηκε σε Eclipse με χρήση του Visual Editor για το γραφικό περιβάλλον.
