Μαθαίνοντας Go


Μια ιστοσελίδα όπως η δική μας που φιλοξενεί ένα παιχνίδι δεν θα μπορούσε να θεωρείται ολοκληρωμένη αν δεν παρείχε και έναν τομέα στον οποίο θα περιλαμβάνονταν εγχειρίδια εκμάθησης του παιχνιδιού. Εδώ λοιπόν σας παραθέτουμε δύο οδηγούς έναν ελληνικό και έναν αγγλικό ώστε να κάνετε τα πρώτα σας βήματα στον κόσμο του Go.

Η ιστοσελίδα μας ωστόσο περιέχει και το online οπτικό εγχειρίδιο σε μορφή java applet το οποίο και θα βρείτει στον τομέα applet της σελίδας.



Ελληνικό εγχειρίδιο
The way to Go



Από τα δύο παραπάνω έγγραφα το Ελληνικό εγχειρίδιο ανήκει στον δικτυακό τόπο European Go Federation (Greece) και είναι ελεύθερο προς διάδοση εφόσον κρατείται αυτούσιο.
Το κείμενο The way to Go αποτελεί ένα από τα πιο αντιπροσωπευτικά ξενόγλωσσα εγχειρίδια και επιτρέπεται η διακίνησή σου με τον ίδιο τρόπο όπως και του Ελληνικού εγχειριδίου. Το συγκεκριμένο αρχείο υπάρχει στο Internet Κατευθείαν διαθέσιμο σε μορφή pdf μέσω του Google.
