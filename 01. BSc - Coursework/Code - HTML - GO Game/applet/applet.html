Οπτικό Εγχειρίδιο

(java applet)


Σε αυτό το σημείο με υπερηφάνια σας παρουσιάζουμε με υπερηφάνια το δικό μας οπτικό εγχειρίδιο εκμάθησης σε μορφή java applet. παρακάτω (αλλά και στον ftp server μας μπορείτε να βρείτε τον πηγαίο κώδικα αλλά και το εκτελέσιμο (.class) αρχείο.

Παρακαλούμε να είστε προσεκτικοί με τη χρήση τόσο του πηγαίου κώδικα όσο και του εκτελέσιμου αρχείου. Θα παρακαλούσαμε να μας ενημερώσετε για οποιαδήποτε αλλαγή θέλετε να κάνετε ή αν ενδεχομένως θέλετε να φιλοξενήσετε το applet στην ιστοσελίδα σας, και να μην προχωρήσετε σε αλλαγές χωρίς την άδειά μας.

Αν κάποιος από εσάς επιθυμεί η σελίδα μας να φιλοξενήσει το δικό του applet για το παιχνίδι Go μπορεί να επικοινωνήσει μαζί μας μέσω e-mail και εμείς θα φροντίσουμε να ανεβάσουμε τη δουλειά του γνωστοποιώντας φυσικά στο κοινό ότι το έργο ανήκει σε αυτόν.
