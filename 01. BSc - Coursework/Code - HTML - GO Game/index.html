Το Go είναι ένα κλασικό επιτραπέζιο παιχνίδι δύο ατόμων, εφάμιλλο με το σκάκι. Διαφέρει βέβαια από αυτό στο γεγονός ότι η ορολογία (του go) δεν παραπέμπει σε πόλεμο μεταξύ δύο στρατών. Αντιθέτως (όσον αφορά τους παλαιότερους όρους που έχουν διασωθεί) αναφέρεται κυρίως σε κινήσεις χεριών (block - μπλοκάρω - , push - σπρώχνω - , throw - πετάω - , κλπ).

Οι αντίπαλοι παίκτες τοποθετούν μαύρα και λευκά πλακίδια στις τομές των γραμμών μια πλάκας (όπως αυτή φαίνεται στα αριστερά της παρούσας σελίδας) που παριστάνει ένα πλέγμα (όπως η σκακιέρα) του οποίου οι διαστάσεις ποικίλουν ανάλογα με το πρότυπο παιχνιδιού που οι παίκτες ακολουθούν (αναλυτικότερη αναφορά για τα διάφορα πρότυπα όπως αυτά έχουν επικρατήσει και διαμορφωθεί σήμερα γίνεται στον τομέα της ιστορίας που φιλοξενεί η ιστοσελίδα μας).

Στόχο του παιχνιδιού αποτελεί η κατάληψη όσο το δυνατόν μεγαλύτερης περιοχής (οι τρόποι με τους οποίους μπορεί κάτι τέτοιο να επιτευχθεί γίνονται περισσότερο κατανοητοί στον τομέα εκμάθησης της ιστοσελίδας μας).

Το go αποτελεί ένα παιχνίδι που απαιτεί από τον παίκτη να είναι συγκεντρωμένος γιατί τα πάντα μπορούν να ανατραπούν με μία λάθος κίνηση. Είναι ένα παιχνίδι τακτικής, αντίληψης και διορατικότητας, στοιχεία που απαιτούν καθαρό μυαλό κατά τη διάρκεια του παιχνιδιού. Σε αυτήν την ιστοσελίδα προσπαθούμε να σας πείσουμε για την ομορφιά και αξία αυτού του παιχνιδιού. Σας παρέχουμε επίσης διάφορους τρόπους εκμάθησης του παιχνιδιού (είτε μέσω κάποιου εγχειριδίου - σας παρέχουμε τα δύο πιο γνωστά που κυκλοφορούν - καθώς επίσης και έναν περισσότερο οπτικό τρόπο σε μορφή java applet).

Το go είναι ένα παιχνίδι που μαθαίνεται σχετικά γρήγορα αλλά χρειάζεστε αρκετή εξάσκηση για την απόκτηση της εμπειρίας που μπορεί να σας κάνει παίκτες υψηλού επιπέδου.

Εμείς από την πλευρά μας σας ευχόμαστε καλή διασκέδαση!
