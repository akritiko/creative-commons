Ευελπιστώντας η ιστοσελίδα μας να σας φάνηκε ενδιαφέρουσα και να σας πείσαμε για την αξία του Go, σαν έναν τελευταίο τομέα παραθέτουμε την Επικοινωνία. Εδώ θα σας δώσουμε λίγα περισσότερα στοιχεία για εμάς.

Σας υπενθυμίζουμε ότι μπορείτε να επικοινωνήσετε μαζί μας για οποιοδήποτε απορία έχετε. Θα είναι χαρά μας να ακούσουμε τις ιδέες σας, να σας προμηθεύσουμε με κάποιο υλικό ή να σας παραπέμψουμε στη σωστή κατεύθυνση. Ακόμη θα ήταν χαρά μας να συνεργαστούμε μαζί σας περιλαμβάνοντας στη σελίδα μας δικά σας κείμενα που αφορούν το Go, διορθώσεις στα ήδη υπάρχοντα (αν παρατηρήσατε κάποιο λάθος) και γιατί όχι το δικά σας applet ή γενικότερα λογισμικό που μπορεί να βοηθήσει τους επισκέπτες να μάθουν την τέχνη του Go.
