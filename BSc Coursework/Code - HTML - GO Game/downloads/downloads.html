GoGame (igowin)


Τονίζουμε ότι η εφαρμογή που παραθέτουμε μπορεί να δημοσιεύεται ελεύθερα από τρίτους αρκεί να μην έχει υποστεί συνάρμοση. Για περισσότερες πληροφορίες μπορείτε να επισκευθείται τον δικτυακό τόπο Greek Go Association παραπομπή του οποίου βρίσκεται στον αντίστοιχο τομέα της σελίδας μας.

Εκτός της συγκεκριμένης εφαρμογής σε αυτό το σημειό παραθέτουμε και τα αρχεία που σχετίζονται με το java applet - tutorial που κατασκευάστηκε από την ομάδα μας. Είστε ελεύθεροι να τα κατεβάσετε ωστόσο θα είμασταν ευγνώμονες αν για οποιαδήποτε επέμβαση ή χρηση αυτών επικοινωνούσατε μαζί μας για να σας δώσουμε την άδειά μας.
