Παραδοτέο για την δεύτερη εργασία στα κατανεμημένα συστήματα και διαδίκτυο.

Το CD αποτελείται από:

1. Πηγαίος κώδικας: περιέχει τον πηγαίο κώδικα της εφαρμογής.
2. Εκτελέσιμο: περιέχει το εκτελέσιμο αρχείο της εφαρμογής.
3. Τεκμηρίωση: περιέχει το τεχνικό έγγραφο.

ΣΧΟΛΙΑ: 

Για την εκκίνηση της εφαρμογής εκτελούμε το Simulate.bat που βρίσκεται στον φάκελο ΕΚΤΕΛΕΣΙΜΟ.

Για την ορθή λειτουργία της εφαρμογής συνίσταται η αντιγραφή του φακέλου ΕΚΤΕΛΕΣΙΜΟ στον σκληρό δίσκο του χρήστη. Αυτό γιατί η εφαρμογή πρέπει να μπορεί να δημιουργήσει ένα αρχείο και το CD δεν προσφέρει δυνατότητες εγγραφής.
